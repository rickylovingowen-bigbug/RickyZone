## 第四部分：武侠小说人物管理系统（v2.5）

### 4.3.8 导入导出（v2.5）

**格式限定**：
- 导入：仅支持 **.xlsx** 格式
- 导出：仅支持 **.xlsx** 格式

**按钮位置**：人物管理界面右上角，与「新建人物」按钮同区域

```
右上角操作区：
[导出 Excel] [导入 Excel] [+ 新建人物]
```

---

### 4.3.3 门派筛选（v2.5）

**默认状态**：筛选面板收起，仅显示「门派筛选」按钮

**展开交互**：
- 点击「门派筛选」按钮展开下拉面板
- 面板包含：搜索框 + 门派列表（按拼音排序）
- 支持多选，实时刷新

**收起交互**：
- 再次点击按钮
- 或点击面板外部区域

---

### 4.3.5 人物卡片（v2.5）

**卡片比例**：竖版扑克牌式，160px × 240px（2:3）

**边框样式**：统一使用 border-bg-tertiary，不根据门派变色

**门派标签**：
- 位置：卡片内部顶部，与 Rank 标签并排
- 样式：圆角标签，背景色标识门派
- 文字颜色：自动适配（深色背景白字，浅色背景黑字）

**信息层级**：
```
┌─────────────────────┐
│ [Rank] [门派标签]    │
│                     │
│      人物姓名        │
│                     │
│ 性别：男  年龄：25   │
│ 职位：掌门           │
│ 兵器：倚天剑         │
│                     │
│    [编辑] [删除]     │
└─────────────────────┘
```
