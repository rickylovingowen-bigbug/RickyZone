## 第四部分：武侠小说人物管理系统（v2.7）

### 4.2 人物数据模型（v2.7 更新）

#### 4.2.1 属性定义（v2.7 更新）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 唯一标识 |
| name | string | 是 | 人物姓名 |
| gender | enum | 否 | 性别: male/female |
| age | number | 否 | 年龄 |
| sect | string | 否 | 所属门派 |
| sectPosition | string | 否 | 门派职位 |
| faction | string | 否 | 所属势力 |
| factionPosition | string | 否 | 势力职位 |
| weapon | string | 否 | 兵器 |
| martialLevel | enum | 否 | 武学 Rank |
| martialTier | enum | 否 | 武学品级（自动计算）|
| isVip | boolean | 否 | VIP 状态（v2.7 新增）|
| appearance | text | 否 | 样貌描述 |
| personality | text | 否 | 性格核心 |
| value | text | 否 | 存在价值 |
| conflict | text | 否 | 主要冲突 |

### 4.3 功能模块（v2.7 更新）

#### 4.3.5 人物卡片（v2.7 更新）

##### 4.3.5.1 卡片尺寸（v2.7 更新）
- **设计目标**：3 张卡片刚好填充一行
- **尺寸计算**：
  - 原尺寸：160px × 240px
  - 新尺寸：约 240px × 360px（等比例放大 1.5 倍）
- **字体比例**：所有字体按 1.5 倍放大

##### 4.3.5.2 VIP 功能（v2.7 新增）

**VIP 标识显示**：
- 位置：人物姓名右侧
- 样式：红色「VIP」标签
- 尺寸：比名字略小
- 显示条件：`isVip === true`

**VIP 编辑**：
- 位置：新建/编辑人物表单
- 组件：开关按钮（Toggle）
- 文案：「VIP 状态」

##### 4.3.5.3 武学品级显示（v2.7 修复）
- 人物卡片上的武学品级应根据 Rank 自动计算并显示
- 不再显示「未知」

##### 4.3.5.4 信息展示顺序（v2.7 更新）

从上到下依次为：

| 顺序 | 信息项 | 显示规则 |
|------|--------|---------|
| 1 | 武学品级 | 自动根据 Rank 计算 |
| 2 | 武学 Rank | 如为空，显示「未知」 |
| 3 | 所属门派 | 如为空，显示「未知」 |
| 4 | 性别 | 如为空，显示「未知」 |
| 5 | 年龄 | 如为空，显示「未知」 |
| 6 | 职位 | 如为空，显示「未知」 |

---

#### 4.3.3 筛选栏（v2.7 更新）

##### 4.3.3.5 VIP 筛选（v2.7 新增）

- **位置**：筛选栏，年龄排序按钮旁边
- **按钮样式**：
  - 未选中：普通按钮，显示「VIP」
  - 选中：红色背景，白色文字，显示「VIP」
- **交互**：点击切换选中状态，实时刷新列表
